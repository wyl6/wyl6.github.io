python函数式编程sorted

#python函数式编程之sorted用法

sorted(iterable,key,reverse) 对所有可迭代的对象进行排序，返回一个list.
sorted主要用在list和dict中
iterable是一个可迭代对象
key制定排序规则，可以是一个函数，比如abs；也可以是指定的参数的某一个元素，参数来自可迭代对象中
reverse表示排序规则，reverse = True降序，reverse = False升序（默认）
